# Professional Portfolio Website

Proyek desain ulang dan implementasi website portfolio profesional responsif menggunakan pendekatan kombinasi utilitas **Tailwind CSS**, struktur modular semantik, dan logika native **Vanilla JavaScript**.

## Fitur Utama
* **Akses Responsif Penuh**: Dioptimalkan untuk perangkat mobile, tablet, hingga desktop monitor lebar.
* **Mode Gelap/Terang Terintegrasi**: Sinkronisasi otomatis berdasarkan preferensi browser dan penyimpanan lokal (*Local Storage*).
* **Optimasi Performa**: Menggunakan teknik *Lazy Loading* bawaan melalui API `IntersectionObserver` untuk aset gambar galeri.
* **Validasi Formulir Sisi Klien**: Perlindungan integritas data masukan sebelum pengiriman data.

## Struktur Direktori
Sesuai dengan ketentuan standar pengembangan perangkat lunak, struktur manajemen file dirancang sebagai berikut:
* `assets/css/custom.css` - Animasi kustomisasi keyframes.
* `assets/js/script.js` - Seluruh pengendali logika interaktivitas & arsitektur DOM.
* `assets/images/` - Penyimpanan aset visual multimedia.
